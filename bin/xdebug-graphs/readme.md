# XDebug Profile Graph Generator

Converts Xdebug cachegrind profiling files into visual PDF call graphs, with Blockera-related functions highlighted.

## Prerequisites

### 1️⃣ Install Python dependencies

```bash
pip install gprof2dot
```

### 2️⃣ Install Graphviz

**macOS:**
```bash
brew install graphviz
```

**Ubuntu/Debian:**
```bash
sudo apt install graphviz
```

## Installation

### 3️⃣ Create the bin directory

```bash
mkdir -p ~/bin
```

### 4️⃣ Create symbolic link (recommended)

This keeps the command synced with the repo:

```bash
ln -sf /Users/aliaghdam/Sites/site-blockera/wp-content/plugins/blockera/bin/xdebug-graphs/xdebug-graphs ~/bin/xdebug-graphs
chmod +x /Users/aliaghdam/Sites/site-blockera/wp-content/plugins/blockera/bin/xdebug-graphs/xdebug-graphs
```

**Alternative:** Copy the file manually:
```bash
cp /path/to/xdebug-graphs ~/bin/xdebug-graphs
chmod +x ~/bin/xdebug-graphs
```

### 5️⃣ Add ~/bin to PATH

Add to `~/.zshrc`:
```bash
export PATH="$HOME/bin:$PATH"
```

Reload:
```bash
source ~/.zshrc
```

## Usage

### Basic usage

**Important:** run from the folder that contains `cachegrind.*` files, or pass that path explicitly.

Use current directory (generates all types):
```bash
cd /path/to/xdebug-profiles
xdebug-graphs
```

Or pass a directory:
```bash
xdebug-graphs /path/to/xdebug-profiles
```

If you see `✅ Done` with no `→ Generating` lines, the tool found no `cachegrind.*` files in that directory (wrong folder), or dependencies are missing (`pip install gprof2dot`, `brew install graphviz`).

### Generate specific types

Available types: `extra-small`, `small`, `medium`, `full`

| Type | Node Threshold | Edge Threshold | Use Case |
|------|----------------|----------------|----------|
| `extra-small` | 5% | 3% | Quick overview, main hotspots only |
| `small` | 2% | 1% | Balanced view |
| `medium` | 0.25% | 0.1% | Detailed analysis |
| `full` | 0.01% | 0.01% | Complete call graph |

Generate only specific types:
```bash
xdebug-graphs --types extra-small,small
xdebug-graphs /path/to/xdebug-profiles --types medium,full
xdebug-graphs --types small
```

If `--types` is not specified, all four types will be generated by default.

## Output

The tool generates PDF files with:
- **Blue highlighted nodes**: Blockera-related functions
- **Red bordered nodes**: Top 5 most expensive Blockera functions
- **Blue edges**: Calls originating from Blockera code
